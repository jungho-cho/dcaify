# DCAify — 암호화폐/투자 DCA 계산기

> 앱 이름: **DCAify**
> 목표: 구글 광고(AdSense + AdMob) 수익 월 200유로
> 타겟: 글로벌 (영어, 암호화폐/투자 관심자)
> 플랫폼: 웹 (우선) + 모바일 앱 (이후 확장)

---

## 핵심 개념

"DCA(Dollar Cost Averaging) 전략으로 얼마나 벌었는지, 앞으로 얼마나 벌 수 있는지 바로 계산해주는 도구"

- 과거 DCA 수익 계산 ("비트코인에 매달 $100씩 3년 투자했으면?")
- 미래 DCA 시뮬레이션 ("앞으로 2년 동안 매달 $200씩 넣으면?")
- 손익분기점 계산
- 세금 계산 (국가별 양도소득세 추정)

---

## 핵심 기능 (MVP)

### 1. DCA 수익 계산기 (메인)
- 코인 선택 (BTC, ETH, SOL 등 주요 코인)
- 투자 시작일 / 종료일
- 투자 주기 (매일 / 매주 / 매달)
- 투자 금액
- 결과: 총 투자금, 현재 가치, 수익률, 수익금

### 2. DCA 시뮬레이터 (미래 예측)
- 현재가 기준 미래 DCA 시뮬레이션
- 목표 가격 설정 → 예상 수익 계산
- 시나리오 비교 (보수/중립/낙관)

### 3. 손익분기점 계산기
- 평균 매수가 입력 → 수익 실현 가격 계산
- 세금 반영 손익분기점

### 4. 세금 계산기 (보너스)
- 국가 선택 (미국/한국/독일/영국 등)
- 단기/장기 보유 세율 자동 적용
- 실수령 수익 계산

### 5. 광고
- 계산 결과 하단 디스플레이 광고 (AdSense)
- 페이지 사이드바 광고
- 모바일 앱 전환 시 AdMob 배너/인터스티셜

---

## 수익 목표 계산

| 지표 | 목표값 |
|------|--------|
| 월 페이지뷰 | 15,000 |
| AdSense RPM | $12-20 (금융 니치) |
| 월 예상 수익 | $180-300 (≈ 170-280유로) |

- 금융 니치는 RPM이 높아 트래픽이 적어도 수익 달성 가능
- 암호화폐 시장 활황기엔 트래픽 2-5배 폭발 → 연평균으로 목표 달성 용이
- 웹 먼저 수익 검증 후 앱으로 확장 시 AdMob 추가 수익

## 데이터 소스

| API | 용도 | 비용 |
|-----|------|------|
| [CoinGecko API](https://www.coingecko.com/en/api) | 코인 과거 가격 데이터 | 무료 tier (10,000 req/월) |
| [CoinCap API](https://coincap.io/) | 실시간 가격 | 무료 |
| [Binance Public API](https://binance-docs.github.io/apidocs/) | 과거 OHLCV 데이터 | 무료 |

- CoinGecko 무료 tier로 MVP 충분
- 트래픽 증가 시 응답 캐싱(Redis/Vercel KV)으로 API 요청 최소화
- 계산 로직 자체는 클라이언트사이드 → 서버 비용 거의 없음

## 기술 스택 (추천)

| 영역 | 선택 | 이유 |
|------|------|------|
| 프레임워크 | Next.js | SEO 필수 (SSR/SSG), Vercel 무료 배포 |
| UI | Tailwind CSS + shadcn/ui | 디자이너 없이 깔끔한 UI 가능 |
| 차트 | Recharts 또는 Chart.js | 수익 그래프 시각화 |
| 광고 | Google AdSense | 금융 니치 고RPM |
| 배포 | Vercel (무료 tier) | 서버 비용 0원 |
| 캐싱 | Vercel KV 또는 로컬 캐시 | API 요청 절약 |

## SEO 전략 (핵심)

트래픽이 수익을 결정하므로 SEO가 가장 중요.

**타겟 키워드:**
- "bitcoin DCA calculator" (월 검색량 ~40,000)
- "crypto DCA calculator" (월 검색량 ~30,000)
- "ethereum DCA calculator" (월 검색량 ~15,000)
- "DCA crypto profit calculator"
- "[코인명] dollar cost averaging calculator"

**페이지 전략:**
- `/` — 메인 (BTC DCA 계산기)
- `/[coin]` — 코인별 전용 페이지 (btc, eth, sol, bnb 등)
- `/blog` — "DCA 전략이란?", "비트코인 DCA 10년 결과" 등 콘텐츠
