# SEO Phase 1 — Index Hygiene Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Reduce DCAify's Google-indexable URL count from ~340 to ~54 high-signal URLs, unify `www`/non-`www` hosts, and emit a clean sitemap — so Google re-evaluates the site as higher quality. No new content in this phase.

**Architecture:** A single source-of-truth policy module (`src/lib/seo.ts`) defines Top-10 coins and Top-5 comparison pairs. Every page's `generateMetadata` consults this module to emit `robots: { index, follow }`. The sitemap emits only indexable URLs. A Next.js middleware issues a 301 from `www.dcaify.com` to `dcaify.com`.

**Tech Stack:** Next.js 16 App Router, TypeScript, Vitest, OpenNext Cloudflare adapter.

**Source spec:** [`docs/superpowers/specs/2026-04-16-seo-indexation-recovery-design.md`](../specs/2026-04-16-seo-indexation-recovery-design.md)

**Indexation rules:**
| Page type | Slug rule | Robots |
|---|---|---|
| Coin page (EN + KO) | slug ∈ Top 10 | index, follow |
| Coin page (EN + KO) | slug ∉ Top 10 | noindex, follow |
| Guide page (EN + KO) | slug ∈ Top 10 | index, follow |
| Guide page (EN + KO) | slug ∉ Top 10 | noindex, follow |
| Comparison page | slug ∈ Top 5 pairs | index, follow |
| Comparison page | slug ∉ Top 5 pairs | noindex, follow |
| KO tax page | any | noindex, follow |
| Home / about / privacy / blog | any | default (index) |

```ts
/**
 * Indexation policy source of truth.
 *
 * Only pages in these lists are indexable by Google. All other page
 * variants emit `noindex, follow` so link equity still flows but the
 * URL does not enter Google's index. See
 * docs/superpowers/specs/2026-04-16-seo-indexation-recovery-design.md.
 */

export const TOP_COIN_SLUGS = [
  'btc',
  'eth',
  'sol',
  'xrp',
  'ada',
  'doge',
  'bnb',
  'avax',
  'dot',
  'link',
] as const

export const TOP_COMPARISON_SLUGS = [
  'btc-vs-eth',
  'btc-vs-sol',
  'btc-vs-doge',
  'eth-vs-sol',
  'doge-vs-eth',
] as const

const TOP_COIN_SET: ReadonlySet<string> = new Set(TOP_COIN_SLUGS)
const TOP_COMPARISON_SET: ReadonlySet<string> = new Set(TOP_COMPARISON_SLUGS)

export function isTopCoin(slug: string): boolean {
  return TOP_COIN_SET.has(slug)
}

export function isTopComparison(slug: string): boolean {
  return TOP_COMPARISON_SET.has(slug)
}

export type PageType = 'coin' | 'guide' | 'comparison' | 'tax'

export interface RobotsDirective {
  index: boolean
  follow: boolean
}

export function shouldIndex(type: PageType, slug: string): RobotsDirective {
  if (type === 'tax') return { index: false, follow: true }
  if (type === 'comparison') {
    return { index: isTopComparison(slug), follow: true }
  }
  // 'coin' and 'guide' share the same Top 10 rule
  return { index: isTopCoin(slug), follow: true }
}
```

import { MetadataRoute } from 'next'
import { TOP_COIN_SLUGS, TOP_COMPARISON_SLUGS } from '@/lib/seo'

export default function sitemap(): MetadataRoute.Sitemap {
  const base = 'https://dcaify.com'
  const now = new Date()

  const pages: MetadataRoute.Sitemap = [
    // Homepage + KO homepage
    { url: base, lastModified: now, changeFrequency: 'weekly', priority: 1 },
    { url: `${base}/ko`, lastModified: now, changeFrequency: 'weekly', priority: 0.9 },

    // Static pages (EN + KO)
    { url: `${base}/about`, lastModified: now, changeFrequency: 'monthly', priority: 0.3 },
    { url: `${base}/ko/about`, lastModified: now, changeFrequency: 'monthly', priority: 0.3 },
    { url: `${base}/privacy`, lastModified: now, changeFrequency: 'monthly', priority: 0.2 },
    { url: `${base}/ko/privacy`, lastModified: now, changeFrequency: 'monthly', priority: 0.2 },
    { url: `${base}/blog`, lastModified: now, changeFrequency: 'weekly', priority: 0.6 },
    { url: `${base}/ko/blog`, lastModified: now, changeFrequency: 'weekly', priority: 0.6 },
  ]

  // Top 10 coin pages (EN + KO)
  for (const slug of TOP_COIN_SLUGS) {
    pages.push(
      { url: `${base}/${slug}`, lastModified: now, changeFrequency: 'weekly', priority: 0.9 },
      { url: `${base}/ko/${slug}`, lastModified: now, changeFrequency: 'weekly', priority: 0.9 },
    )
  }

  // Top 10 guide pages (EN + KO)
  for (const slug of TOP_COIN_SLUGS) {
    pages.push(
      { url: `${base}/${slug}/guide`, lastModified: now, changeFrequency: 'monthly', priority: 0.7 },
      { url: `${base}/ko/${slug}/guide`, lastModified: now, changeFrequency: 'monthly', priority: 0.7 },

  // Top 5 comparison pairs (EN only — no KO comparison routes)
  for (const pair of TOP_COMPARISON_SLUGS) {
    pages.push(
      { url: `${base}/${pair}`, lastModified: now, changeFrequency: 'monthly', priority: 0.5 },

- [ ] **Step 4: Implement `src/middleware.ts`**

```ts
import { NextRequest, NextResponse } from 'next/server'

export function middleware(request: NextRequest) {
  const host = request.headers.get('host') ?? request.nextUrl.host

  if (host.toLowerCase().startsWith('www.')) {
    const apexUrl = new URL(request.nextUrl.toString())
    apexUrl.host = host.slice(4)
    apexUrl.protocol = 'https:'
    return NextResponse.redirect(apexUrl, 301)
  }

  return NextResponse.next()
}

export const config = {
  // Run on every request except Next.js internals and static assets.
  matcher: ['/((?!_next/|favicon.ico|.*\\..*).*)'],
}
```

Notes on design:
- We read `host` directly rather than from `nextUrl.host` because `NextRequest`'s URL can be constructed from either, and the header is the ground truth for what Cloudflare routed to us.
- We explicitly set `protocol = 'https:'` so that any rare http www hit gets upgraded in the same 301.
- The matcher excludes `_next/` and any path with a file extension so we don't redirect static asset fetches unnecessarily.
